Удобный доступ к Яндекс Словарям. Позволяет переводить слова через адресную строку (omnibox).

★ Поддержка почти всех комбинаций языков Яндекс Словарей
★ Изменяемая пара языков по-умолчанию
★ Возможность использовать дополнительные пары языков

Чтобы перевести слово:
1. Выделите адресную строку (ctrl+l / ⌘+l).
2. Напишите y слово_которое_хотите_перевести
3. (по желанию) выберите перевод из предложенных вариантов
4. (по желанию) нажмите [enter], чтобы открыть слово на Яндекс Словарях
